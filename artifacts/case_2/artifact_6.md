# Модель данных и БД

<div class="alert alert-light border">
  Здесь мы перенесём структуру из твоего документа “Описание таблиц БД” в компактный портфолио-вид:

## Основные сущности (черновой список)
- Хозяйство / пользователи / роли
- Поля / культуры / границы (геометрия)
- Техника (комбайны, БП, зерновозы, др.)
- Смены / сменное задание
- Маршрут / точки / трек / полосы
- События (статусы, поломки, задержки)
- Погода (оповещения/факторы перепланирования)
